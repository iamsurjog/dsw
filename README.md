# DBMS Project Backend
## Installation
To download the repo run
```bash
 git clone https://github.com/iamsurjog/dsw.git
 py -m venv ./.venv
 pip install -r requirements.txt
```

## Running it
### Windows
```bash
.\.venv\Scripts\activate
flask --app app run
```

### Linux
```bash
source .venv/bin/activate
flask --app app run
```
